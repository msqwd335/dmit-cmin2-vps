## DMIT 是什么，为什么 CMIN2 线路值得关注

DMIT 是一家专注于亚太区优质线路的 VPS 服务商，总部在美国，主要卖点是针对中国大陆用户优化的网络线路。他们的产品线里有 CN2 GIA、CMIN2、以及普通 BGP 线路，覆盖香港、日本、洛杉矶等节点。

CMIN2 是中国移动的国际精品网络，全称 China Mobile International Network2。和 CN2 GIA（电信）相比，CMIN2 对移动用户的体验更好，联通和电信也有一定优化，属于三网都能用、移动用户特别受益的那种线路。

我自己测过香港 CMIN2 节点，晚高峰延迟稳定在 30ms 左右，丢包率很低。对于需要稳定连接国内的场景，这个表现是真的够用。

## 适合哪些人用

- 主力网络是中国移动的用户，CMIN2 对移动的优化是最直接的
